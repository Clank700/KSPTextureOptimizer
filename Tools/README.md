# Tools

Put `texconv.exe` here if you want local DDS conversion without adding it to `PATH`.

Recommended install:

```powershell
winget install Microsoft.DirectXTex.Texconv
```

Alternative local layout:

```text
KSPTextureOptimizer\Tools\texconv.exe
```

`texconv.exe` is from Microsoft DirectXTex, which is MIT licensed:

https://github.com/microsoft/DirectXTex
