# Security and Safety

This is a local Windows desktop utility that reads and writes files selected by the user.

Report issues if the tool:

- Writes outside the selected KSP/mod folder, backup folder, staging folder, or run manifest folder.
- Restores a backup over a file that changed after optimization without warning.
- Optimizes while `KSP_x64.exe` is running.
- Misidentifies unsupported formats as safe.

For personal use, keep backups until you have tested several KSP scenes.
